# T-Link
## What is it? ##
It's a family location sharing app that tracks both phone locations, and car locations using the Tesla Fleet API. This app was built dedicated to my family and is not necessarily _intended_ for public use.
## Code Structure
This Github repository is only the backend code; this requires a frontend too. All files and folders from this repository are required to run the backend except the setup folder. This setup folder was created during the development process of this service, and is technically no longer needed unless another person wants to set this up for themself.
